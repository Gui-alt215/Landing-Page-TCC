# Landing-Page-TCC

O AgendAí é um projeto que reproduz um sistema de agendamento de consultas para clínicas de estética, proporcionando uma interface moderna, organizada e de fácil visualização.  Ele demonstra de forma prática e eficiente como seria a gestão de horários, serviços e atendimentos, servindo como suporte para apresentações, portfólio ou TCC.  Os membros Guilherme Almeida Silva Ferreira, Gabriel do Nascimento Moreira da Silva, Gabriely Santos Lopes, Guilherme Gomes Portugal e Higor Sousa Morato foram responsáveis pelo desenvolvimento do projeto.
O AgendAí oferece uma interface intuitiva, mostrando a agenda do dia, o status dos atendimentos, os horários confirmados, informações sobre os serviços e a organização por salas ou profissionais.  Ele demonstra como um software poderia centralizar e visualizar as marcações, reduzir os desencontros na recepção e otimizar o fluxo de atendimentos em clínicas de estética.
Este projeto é valioso porque mostra na prática como a automação pode melhorar a gestão de horários em clínicas de estética.  Ele diminui erros de agendamento, torna as tarefas da recepção mais simples, melhora a experiência do cliente e organiza a rotina dos profissionais.  O AgendAí serve como uma prova visual e estruturada de como um sistema digital pode otimizar processos, reduzir faltas e aumentar a eficiência da clínica no trabalho de conclusão de curso.
Os usuários podem acessar o site diretamente usando o link fornecido no GitHub Pages, sem necessidade de instalação.  Para visualizar o projeto, é só colocar o endereço no navegador.  Se quiserem modificar o código ou executá-lo localmente, podem clonar o repositório usando o comando git clone e abrir a pasta no VS Code, ou apenas abrir o arquivo index.html no navegador.  O projeto não exige dependências ou instalações adicionais, pois utiliza apenas HTML, CSS e JavaScript, sendo totalmente compatível com qualquer navegador atual.  Ademais, o AgendAí pode ser facilmente personalizado, possibilitando alterações no texto, nas cores, no layout ou no conteúdo da agenda simulada.
